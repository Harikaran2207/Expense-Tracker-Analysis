# Expense Tracker Analysis 📊

## Overview

This project analyzes personal expense data using Python. The goal is to understand spending habits, identify major expense categories, and visualize financial trends through charts.

## Objectives

- Analyze spending patterns
- Identify the highest spending category
- Compare payment methods
- Visualize daily expense trends
- Generate meaningful insights from data

## Technologies Used

- Python
- Pandas
- Matplotlib
- VS Code

## Dataset

The dataset contains:

Column| Description
Date| Transaction date
Category| Expense category
Amount| Expense amount (₹)
Mode| Payment method

## Categories:

- Food
- Travel
- Shopping
- Entertainment
- Miscellaneous

## Payment Modes:

- UPI
- Cash
- Card

## Features

Data Analysis

- Total spending calculation
- Average transaction value
- Category-wise spending analysis
- Payment mode analysis
- Top transactions identification

Data Visualization

- Bar chart for category spending
- Pie chart for expense distribution
- Line chart for daily spending trend

##  Results

- Total Transactions: 45
- Total Spending: ₹12,155
- Highest Spending Category: Shopping
- Average Transaction Value: ₹270.11
